1. Basic Link Syntax

2. Target Attribute

        <a href="https://www.google.com" target="_blank">click for google</a>
        
        //1.target="_blank": Opens the link in a new tab/window.
        //2.target="_self": Opens in the same tab (default).
        //3.target="_parent" / _top: Used in frames.
click for google

3. Types of Links

        Type	Example
        Absolute Link	<a href="https://google.com">Google</a>
        Relative Link	<a href="/about.html">About Page</a>
        Email Link      <a href="mailto:name@example.com">Email Us</a>
        Phone Link	<a href="tel:+911234567890">Call Us</a>
        Download Link	<a href="file.pdf" download>Download PDF</a>

4. Anchor Links (Jump Links)

Used to jump to a section on the same page.

        
Step 1: Add an id to the target section

        <h2 id="contact">Contact Us:+tel number</h2>
        
Step 2: Create a link to it

        <a href="#contact">Go to Contact Section</a>
        
âœ… Note: This works within the same page, or across pages like:

        <a href="about.html#team">Go to Team Section in About Page</a>
